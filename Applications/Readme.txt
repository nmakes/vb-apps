                                                        MULTIPAD

This is the first version of multipad created by Naveen. Use this program to  write anything or just scribble on a sheet. You can also make use of calculator which can do the following:
1) Add
2) Subtract
3) Multiply
4) Divide
5) Square
6) Square Root

This program can be run on any system. Windows 98/Me/2000/Xp/Vista. For any queries email me to navking.venkat7@gmail.com.
